System tablespace là vùng lưu trữ cho change buffer. Nó cũng có thể chứa dữ liệu bảng và chỉ mục nếu các bảng được tạo trong system tablespace thay vì các tablespace riêng lẻ hoặc tablespace chung.

System tablespace có thể có một hoặc nhiều tệp dữ liệu. Theo mặc định, một system tablespace data file duy nhất, có tên là ibdata1, được tạo trong thư mục dữ liệu. Kích thước và số lượng system tablespace data files được xác định bởi tùy chọn khởi động [`innodb_data_file_path`](https://dev.mysql.com/doc/refman/8.4/en/innodb-parameters.html#sysvar_innodb_data_file_path). Để biết thông tin cấu hình, hãy xem [System Tablespace Data File Configuration](https://dev.mysql.com/doc/refman/8.4/en/innodb-init-startup-configuration.html#innodb-startup-data-file-configuration "System Tablespace Data File Configuration").
