# Cloud SDK

The Google Cloud SDK là một bộ các tool dùng để quản lý tài nguyên, máy chủ ứng dụng trên GC. Các tools bao gồm gcloud, gsutil, bq, thư viện ngôn ngữ người dùng cần thiết, IDE extensions.
